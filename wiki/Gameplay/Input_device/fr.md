# Dispositif d'entrée

Un **dispositif d'entrée** est un dispositif qui est utilisé pour contrôler l'entrée sur un ordinateur ou un logiciel spécifique. Dans le wiki d'osu!, ce terme fait spécifiquement référence aux périphériques qui peuvent être utilisés dans osu!

Voici une liste des périphériques d'entrée les plus courants utilisés dans osu! :

- [Souris](/wiki/Gameplay/Input_device/Mouse)
- [Clavier](/wiki/Gameplay/Input_device/Keyboard)
- [Dispositif tactile](/wiki/Gameplay/Input_device/Touch_device)
- [Tablette graphique](/wiki/Gameplay/Input_device/Graphics_tablet)
- [Wiimote](/wiki/Gameplay/Input_device/Wiimote)
- [Contrôleur de tambour](/wiki/Gameplay/Input_device/Drum_controller)
- [Tapis de danse](/wiki/Gameplay/Input_device/Dance_pad)
- [Contrôleur Beatmania](/wiki/Gameplay/Input_device/Beatmania_controller)
