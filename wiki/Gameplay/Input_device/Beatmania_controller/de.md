# Beatmania-Controller

![](img/beatmania-controller.jpg "Beatmania-Controller mit 7 Tasten")

Ein **Beatmania-Controller** ist eine Art von Game-Controller, der mit der *[Beatmania](https://en.wikipedia.org/wiki/Beatmania)* Spieleserie kompatibel ist.

Aufgrund ihrer Bauart werden Beatmania-Controller am häufigsten als Eingabegeräte in [osu!mania](/wiki/Game_mode/osu!mania) verwendet.

## Eingabemethoden

Spieler, die einen Playstation 2 (PS2) *Beatmania*-Controller besitzen, können den Controller mit einem Adapter über USB an ihren Computer anschließen und die Tasten in den Optionen einstellen. Spieler, die hingegen einen *DJ DAO IIDX* Controller besitzen, können den Controller einfach über USB an ihren Computer anschließen und die Tasten belegen.
