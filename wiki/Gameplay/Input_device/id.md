# Perangkat input

**Perangkat input** atau *input device* adalah sebuah perangkat yang digunakan untuk mengontrol input pada komputer atau bagian-bagian khusus dari perangkat lunak. Istilah ini digunakan pada osu! wiki yang secara khusus mengacu pada perangkat yang dapat digunakan di osu!.

Berikut ini adalah daftar perangkat input yang paling umum digunakan untuk osu!:

- [Keyboard](/wiki/Gameplay/Input_device/Keyboard)
- [Perangkat sentuh](/wiki/Gameplay/Input_device/Touch_device)
- [Graphics tablet](/wiki/Gameplay/Input_device/Graphics_tablet)
- [Wiimote](/wiki/Gameplay/Input_device/Wiimote)
- [Kontroller drum](/wiki/Gameplay/Input_device/Drum_controller)
- [Dance pad](/wiki/Gameplay/Input_device/Dance_pad)
