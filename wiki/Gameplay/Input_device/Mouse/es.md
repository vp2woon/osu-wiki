# Ratón

El **ratón** es una forma muy popular de jugar osu!. Puedes usar el ratón para apuntar y hacer clic. El botón central del ratón está configurado para pausar el juego (se puede desactivar en las opciones) y la rueda de desplazamiento se puede usar para cambiar el volumen.

## Opciones relacionadas con el ratón

### Sensibilidad

Esta opción te permite seleccionar una velocidad de ratón personalizada y está configurada en 1x de forma predeterminada.

### Puntero directo

El puntero directo evitará la aceleración del ratón de Windows y proporcionará el movimiento más preciso del ratón. Es necesario habilitar el puntero directo para configurar la sensibilidad del ratón personalizada en Windows 7+.

### Confinar cursor

Esta opción permite que el cursor del ratón permanezca dentro de la ventana de osu! y está configurado en `Solo en pantalla completa` de forma predeterminada. Las opciones disponibles incluyen:

- `Nunca`: El ratón no se quedará dentro de la ventana de osu!.
- `Solo en pantalla completa`: El ratón permanecerá dentro de la ventana de osu! cuando esté en pantalla completa.
- `Siempre`: El ratón siempre permanecerá dentro de la ventana de osu!.

### Deshabilitar la rueda del ratón mientras juegas

Durante el juego, puedes usar la rueda del ratón para ajustar el volumen y pausar el juego. Habilitar esta opción deshabilitará esta funcionalidad mientras juegas.

### Deshabilitar los botones del ratón mientras juegas

Esta opción desactivará todos los botones del ratón durante el juego, lo que puede resultar útil para las personas que usan el teclado para hacer clic.

### Ondas del cursor

El cursor dejará ondas hacia afuera al hacer clic.
