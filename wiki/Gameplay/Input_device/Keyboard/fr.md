# Clavier

Un **clavier** est le périphérique d'entrée le plus communément utilisé pour jouer à osu!. Par défaut, les touches `Z` et `X` sont configurées pour cliquer, mais elles peuvent être changées en n'importe quelle touche de votre choix dans les options d'osu!

## Pavé numérique

Un **pavé numérique** est un clavier miniature avec un nombre limité de touches. Les pavé numérique sont populaires chez osu! en raison de leur petite taille et du nombre de touches, généralement 2 ou 4.
