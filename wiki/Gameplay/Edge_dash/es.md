# Edge dash

El **edge dash** (también conocido como *pixel dash*) es un término usado en [osu!catch](/wiki/Game_mode/osu!catch) para un espacio muy grande entre dos [objetos activos](/wiki/Beatmapping/Active_object) cuando no se alcanza la [distancia de activación](/wiki/Beatmapping/Trigger_distance) requerida y, como tal, no se genera una [hiperfruta](/wiki/Gameplay/Hit_object/Hyperfruit) para crear un [hyperdash](/wiki/Gameplay/Hyperdash). El primer objeto debe ser atrapado con el borde del plato para poder atrapar el segundo objeto.
