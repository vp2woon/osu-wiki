# 总分

*对于更多义项，参见：[Score (消歧义)](/wiki/Disambiguation/Score)。*

**总分**是玩家游玩所有谱面所得到的分数总和，也包括游玩失败的分数。

玩家首次游玩一张上架谱面并取得成绩，总分和[进榜总分](/wiki/Gameplay/Score/Ranked_score)都会相应增加。如果往后的成绩没有超过最高记录，则只有总分会增加。

## 等级

玩家的等级只由**总分**决定。升一级所需的分数由下面的公式算出：

```
score(n) = 5,000 / 3 * (4n^3 - 3n^2 - n) + 1.25 * 1.8^(n - 60) 如果 n <= 100
score(n) = 26,931,190,827 + 99,999,999,999 * (n - 100)         如果 n > 100
```

介于游戏有四舍五入的机制，升到 100 级所需的总分数值和公式里的并不吻合。这是因为游戏会保留一个列表，里面的元素是预先计算好的每升一级所需的分数值，这些值都会约至整数。譬如，列表中第 81 个元素的值为 `score(81) - score(80) = 128,927,482.36216` ，约至整数得 128,927,482。在 100 级之前，总共有 +1.629 分的偏差，所以 `score(100) = 26,931,190,828.629` 和公式第二部分的 26,931,190,827 对不上。

升到下一级的进度以长条的形式显示在玩家的等级旁边。
