# 10K (mod)

![10K mod icon](/wiki/shared/mods/10K.png "10K mod icon")

*For the full list of all mods, see: [Game Modifier](/wiki/Gameplay/Game_modifier)*\
*Not to be confused with [xK (mod)](/wiki/Gameplay/Game_modifier/xK).*

## About

- Type: Special
- Compatible game modes: ![][osu!mania]

## Description

*Notice: This mod is no longer available for use.*

The **10K** mod was a [game modifier](/wiki/Gameplay/Game_modifier) for [osu!mania](/wiki/Game_mode/osu!mania) that enabled the player to use 10 keys for a single playfield on all *converted* osu!mania [beatmaps](/wiki/Beatmap). It functions exactly the same as the rest of the [xK](/wiki/Gameplay/Game_modifier/xK) mods, and thus was not rankable.

[osu!mania]: /wiki/shared/mode/mania.png "osu!mania"
