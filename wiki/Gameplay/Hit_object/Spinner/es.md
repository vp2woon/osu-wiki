# Spinner

El *spinner* es un [objeto](/wiki/Gameplay/Hit_object) en [osu!](/wiki/Game_mode/osu!) que ocupa toda el área de juego. Para completar un spinner, mantén presionado el botón del ratón/teclado especificado (o mantén el lápiz en la tableta). Desde allí, usa el ratón (o lápiz) y gira el spinner con un movimiento circular (en cualquier dirección) hasta que el spinner crezca completamente hacia afuera. Aparecerá un aviso *Clear* para indicar que se completó el spinner. Si el spinner se completó antes de tiempo, continúa girando para obtener una bonificación adicional de [puntuación](/wiki/Gameplay/Score) y recuperar algo de [salud](/wiki/Gameplay/Health).

El círculo blanco exterior muestra cuánto tiempo queda para completar el spinner. Este círculo se volverá rojo para notificar que el tiempo casi se acaba. Los [beatmaps](/wiki/Beatmap) más antiguos, que usan la *versión de la skin 1.0*, tendrán un medidor para indicar qué tan cerca está el jugador de completar el spinner. El cuadro pequeño debajo del spinner muestra los giros por minuto. El número dentro de ese cuadro se traduce en el número de giros posibles, si continúa girando el spinner a esa velocidad durante un minuto. Si se soltó el botón del ratón/teclado (o el lápiz no tocó la tableta) antes de tiempo durante el giro, el spinner dejará de leer el giro.

El número máximo de giros por minuto que se puede lograr es 477, que también es la velocidad a la que el modificador del juego [Auto](/wiki/Gameplay/Game_modifier/Auto) gira, mientras que el modificador del juego [Spun Out](/wiki/Gameplay/Game_modifier/Spun_Out) gira más lento a 287 giros por minuto.
