# Hyperfruit

Un **hyperfruit** est un effet qui peut être généré sur un [fruit](/wiki/Gameplay/Hit_object/Fruit) ou un [drop](/wiki/Gameplay/Hit_object/Juice_stream#drop) lorsque le prochain fruit ou le prochain drop ne peut pas être attrapé avec un dash normal. Le mouvement qui est créé après avoir attrapé l'hyperfruit est appelé un [hyperdash](/wiki/Gameplay/Hyperdash). Cet effet est également généré lorsque la dernière [banane](/wiki/Gameplay/Hit_object/Banana) d'un [spinner](/wiki/Gameplay/Hit_object/Spinner) ne peut pas être attrapée avec un dash.

Durant la partie, cet effet se caractérise par un contour coloré sur l'hyperfruit.
