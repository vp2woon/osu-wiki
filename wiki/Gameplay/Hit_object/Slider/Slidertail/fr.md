# Slidertail

Un *slidertail* ou **sliderend** désigne la partie finale d'un [slider](/wiki/Gameplay/Hit_object/Slider) où le joueur peut relâcher sa touche enfoncée après avoir suivi un [sliderbody](/wiki/Gameplay/Hit_object/Slider/Sliderbody). Les sliders ne se trouvent que dans les [beatmaps](/wiki/Beatmap) du mode [osu!](/wiki/Game_mode/osu!). Les slidertail peuvent être soit entièrement visibles sous forme de cercle, soit invisibles sous forme de partie du sliderbody, selon la version utilisée par votre [skin](/wiki/Skin). La couleur des slidertails est déterminée par les combo colours utilisée par les mappeurs d'une beatmap.
