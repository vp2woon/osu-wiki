# Taiko Korean Beatmap Contest #1

![Logo du TKBC1](img/logo.png)

Le **Taiko Korean Beatmap Contest #1** (***TKBC1***) était un concours de mapping osu!taiko pour les Coréens. Il s'agissait du premier volet du TKBC.

## Calendrier du concours

| Événement | Horodatage (UTC +9) |
| --: | :-- |
| Phase d'inscription | 21/04/2020/26/04/2020 |
| Phase de soumission | 27/04/2020/10/05/2020 |
| Phase d'évaluation | 11/05/2020/31/05/2020 |
| Annonce des résultats | 01/06/2020 |

## Règlement

### Règlement du concours

- Ce concours concerne le mode osu!taiko.
- Vous devez choisir une musique qui correspond au thème.
- Le drain time ne doit pas être inférieur à 80 ni supérieur à 150.
- Les difficultés doivent être de niveau Oni ou supérieur.
- Si vous soumettez plusieurs maps, la dernière sera votre entrée.
- Vous ne devez pas révéler votre map avant l'annonce des résultats.
- Vous ne pouvez pas soumettre ce que vous avez déjà téléchargé.
- Seuls les Coréens peuvent participer à ce concours.
- Les juges ne peuvent pas participer à ce concours.

### Règles relatives aux métadonnées

- L'artiste doit être `asdf`.
- Le titre doit être `1223`.
- Les difficultés doivent être Oni ou Inner Oni.
- La source et les tags doivent être laissées en blanc.
- Lorsque vous soumettez votre map, incluez les métadonnées dans l'email.

## Thème

![Résultats du thème](img/theme.png)

- Chien / Chat
- Fleur
- Mer

Le thème "Chien / Chat" a été sélectionné par un vote.
