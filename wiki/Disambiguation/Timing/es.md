# Tiempo (desambiguación)

**Tiempo** o *temporización* puede tener múltiples significados:

- El proceso de [determinar la estructura musical de una canción](/wiki/Beatmapping/Timing) para una alineación adecuada de los objetos.
- La [pestaña correspondiente](/wiki/Client/Beatmap_editor/Timing) del editor de beatmaps que contiene ajustes relacionados con el tiempo.
- El ancho de las [ventanas de tiempo](/wiki/Beatmap/Overall_difficulty#tiempo), que rige los [juicios](/wiki/Gameplay/Judgement).
