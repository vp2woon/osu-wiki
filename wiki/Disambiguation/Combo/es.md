# Combo (desambiguación)

**Combo** puede tener múltiples significados:

- El [multiplicador de puntuación activo](/wiki/Gameplay/Combo_(score_multiplier)) visible durante el juego, o el número de objetos completados seguidos sin fallar.
- Una [cadena corta de objetos](/wiki/Beatmapping/Combo) que comparten el mismo [color](/wiki/Beatmapping/Combo_colour)
