# Ouendan (значения)

**Ouendan** может означать:

- В японской культуре — группа поддержки, повышающая боевой дух спортивной команды с помощью музыкальных инструментов, транспарантов и подбадривающих выкриков примерно так же, как это делают чирлидеры.
- Серия [японских игр на Nintendo DS](/wiki/iNiS_games), на геймплее которых основана osu!.
  - Этот же термин иногда используется для обозначения самой первой игры данной серии.
- Обобщённое название для играбельных персонажей из вышеупомянутой серии.
