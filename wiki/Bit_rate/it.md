# Velocità di trasferimento

*Per le norme relative al bit rate, vedere: [Criteri di classificazione](/wiki/Ranking_criteria)*.\
*Per una guida alla ricodifica dell'audio a un bit rate diverso, vedere: [Guida all'editing audio](/wiki/Guides/Audio_editing).*

La **velocità di trasferimento** (o bit rate) è una misura della velocità di trasmissione dei dati di un file multimediale in unità di bit nel tempo. In osu! questo è rilevante quando si parla di qualità dei file audio: un audio di qualità superiore richiede in genere un bit rate più alto, mentre un audio di qualità inferiore può utilizzare un bit rate più basso.

La velocità in bit dei file audio varia in genere da 96kbps (chilobit al secondo) a 320kbps. I [criteri di classificazione](/wiki/Ranking_criteria) richiedono che l'audio delle canzoni sia nell'intervallo 128-192kbps e [AiMod](/wiki/Client/Beatmap_editor/AiMod) avvisa se questo non è il caso.
