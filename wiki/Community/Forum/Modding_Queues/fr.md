# Modding Queues

[**Modding Queues**](https://osu.ppy.sh/community/forums/60) est un forum dédié aux demandes de [review des beatmaps](/wiki/Modding), ou de modding. Il est constitué de fils de discussion, ou *queues*, appartenant à différents moddeurs et mappeurs. Chaque file d'attente a des exigences et des limites individuelles pour l'obtention des mods, qui doivent être respectées par les visiteurs afin de recevoir un mod.

Chaque mode de jeu a son propre sous-forum pour les files d'attente, et osu! occupe la section principale du forum :

- [osu!](https://osu.ppy.sh/community/forums/60)
- [osu!taiko](https://osu.ppy.sh/community/forums/121)
- [osu!catch](https://osu.ppy.sh/community/forums/122)
- [osu!mania](https://osu.ppy.sh/community/forums/123)

## Queue rounds

Pour maintenir leur charge de travail à un niveau approprié, la plupart des moddeurs n'acceptent les demandes que pendant de courtes périodes appelées rounds. Avant le début de chaque tour, le propriétaire de la file d'attente publie une courte annonce, qui contient généralement les éléments suivants :

- Le nombre de beatmaps qui peuvent être acceptées dans ce round.
- Exigences spécifiques au round actuel
- Si les mods sont réguliers (normaux ou *NM*), ou ["mod for mod" (*M4M*)](/wiki/Modding/Mod_for_mod), où un mod en retour est attendu.
- Autres informations relatives au modding

Une fois qu'il y a suffisamment de demandes dans le fil de discussion, le moddeur postera un court message indiquant les beatmaps qui ont été acceptées pour le modding. Après cela, la file d'attente est dite fermée jusqu'au prochain round, pendant que le propriétaire modde les beatmaps demandées.
