# 更新通道

osu! 有若干 **更新通道**（即客户端的版本），这些通道以不同的频率接收更新。为了早点发现潜在问题，新的功能和变化在发布到正式版之前，会先发布到预览版本。

| 更新通道 | 说明 |
| :-- | :-- |
| 正式版 (Stable) | 目前最受欢迎的版本；在候选版之后收到更新；[不再开发新功能（feature-locked）](https://zh.wikipedia.org/wiki/Freeze_(software_engineering)) |
| 候选版 (Beta) | 在测试版之后收到更新；不再开发新功能 |
| 测试版 (Cutting Edge) | 接收最新更新；不再开发新功能 |
| [Lazer](Lazer) | 经过重写的游戏客户端；开发进程最活跃，拥有[许多新的功能和变化](/wiki/Client/Release_stream/Lazer/Feature_comparison)。 |
