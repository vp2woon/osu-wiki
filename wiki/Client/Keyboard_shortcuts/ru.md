# Горячие клавиши

В osu! есть множество горячих клавиш, которые можно использовать как в игре, так и в режиме ожидания на любом игровом экране.

*Примечание: в этой статье приведены стандартные сочетания клавиш, поменять которые можно в [настройках](/wiki/Client/Options).*

## Общие

Эти сочетания работают везде:

| Горячая клавиша | Действие |
| :-- | :-- |
| `Alt` + `F4` | Закрыть игру (без предупреждений). При использовании во время игры, в редакторе или osu!direct вы вернетесь к экран выбора песни, либо (из osu!direct) в главное меню. |
| `Alt` + `Стрелки вверх/вниз ` | Изменить общую громкость игры. Чтобы изменить громкость эффектов и музыки, используйте стрелки влево или вправо. |
| `Ctrl` + `Alt` + `Shift` + `S` | Перезагрузить текущий скин. При нажатии в процессе игры или на экране результатов перезагрузка отложится до возвращения на экран выбора песни. |
| `Ctrl` + `F11` | Спрятать/показать диаграмму рендеринга кадров. Нажатие на `Ctrl` приостановит её. |
| `F7` | Изменить ограничение по количеству кадров. |
| `F8` | Спрятать/показать чат. |
| `F9` | Спрятать/показать чат и пользователей/друзей онлайн. |
| `F10` | Включить/выключить кнопки мыши (кроме колёсика мыши, оно работает всегда). |
| `F12` | Сделать скриншот. Чтобы открыть папку со скриншотами, нажмите на уведомление. |
| `Shift` + `F12` | Сделать скриншот и загрузить его на сервер. Скриншот будет также сохранён на вашем компьютере и откроется в браузере после загрузки. |
| `Insert` | [Клавиша-анти-босс](https://en.wikipedia.org/wiki/Boss_key): сворачивает игру в системный трей и отключает её звук. |
| `Shift` + `F1` | Открыть [osu!status](https://twitter.com/osustatus) в браузере при ошибке подключения. |
| `Ctrl` + `O` | Открыть настройки. Универсальный оффсет можно будет изменить, только находясь в главном меню. |

## Главное меню

| Горячая клавиша | Действие |
| :-- | :-- |
| `Alt` + `Enter` | Включить/выключить полноэкранный режим. |
| `Стрелки вверх/вниз` | Изменить общую громкость игры. Чтобы изменить громкость эффектов и музыки, используйте стрелки влево или вправо. |
| `Esc` | Вернуться на шаг назад. |
| `P`, `E`, или `O` | Открыть меню игры. |
| `P` три раза | Перейти в режим одиночной игры. |
| `P` дважды, затем `M` | Войти в [мультиплеер](/wiki/Client/Interface/Multiplayer). |
| `E` дважды | Открыть редактор. |
| `O` | Открыть настройки. |
| `Q` | Показать диалог выхода из игры. |
| `Esc` | Показать диалог выхода из игры, когда все меню закрыты. |
| `F` | Включить счётчик кадров. |

### Проигрыватель

| Горячая клавиша | Действие |
| :-- | :-- |
| `Z` или `F1` | Воспроизвести предыдущую песню. |
| `X` или `F2` | Воспроизвести песню. Если музыка уже играет, перемотать в начало песни. |
| `C` | Поставить на паузу/воспроизвести. |
| `F4` | Остановить воспроизведение и вернуться в начало. |
| `V` или `F5` | Воспроизвести следующую песню. |
| `J` или `F6` | Открыть список песен. |
| `R` | Воспроизвести случайную песню. |

| `F5` | Переиндексировать карты (может показать диалог). |

| `Ctrl` + `1` | Переключиться в [osu!](/wiki/Game_mode/osu!). |
| `Ctrl` + `2` | Переключиться в [osu!taiko](/wiki/Game_mode/osu!taiko). |
| `Ctrl` + `3` | Переключиться в [osu!catch](/wiki/Game_mode/osu!catch). |
| `Ctrl` + `4` | Переключиться в [osu!mania](/wiki/Game_mode/osu!mania). |
| `Ctrl` + `Enter` | Запустить выбранную карту с модом [Auto](/wiki/Gameplay/Game_modifier/Auto). |
| `Ctrl` + `Shift` + `Enter` | Запустить выбранную карту с модом [Cinema](/wiki/Gameplay/Game_modifier/Cinema). |
| `Ctrl` + `A` | Включить/выключить мод Auto. |
| `Ctrl` + `F3` или `Ctrl` + `+` | Увеличить скорость osu!mania (работает только в режиме osu!mania). |
| `Ctrl` + `F4` или `Ctrl` + `-` | Уменьшить скорость osu!mania (работает только в режиме osu!mania). |
| `Ctrl` + `Backspace` | Удалить последнее слово в поисковом запросе. |
| `Стрелки вверх/вниз` | Выбрать следующую/предыдущую сложность. |
| `Стрелки влево/вправо` | Выбрать следующую/предыдущую карту. |
| `Page Up/Down` | Пролистать на страницу вверх/вниз. |
| `Enter` | Открыть выбранную карту или коллекцию, либо начать играть выбранную сложность. |
| `Shift` + `Стрелки влево/вправо` | Переключиться на следующую/предыдущую группу карт. |
| `Shift` + `Enter` | Раскрыть/скрыть текущую группу карт. |

### Выбор модов

*Основная статья: [Игровые модификаторы](/wiki/Gameplay/Game_modifier)*\
*Примечание: нажатие на клавишу включает или отключает выбранный мод. Пользоваться этими горячими клавишами можно только в меню выбора модов.*\
*Примечание: Target practice (osu!), 1K и 2K (osu!mania), Co-op (osu!mania), а также Mirror (osu!mania) не имеют горячих клавиш.*

| Горячая клавиша | Мод |
| :-- | :-- |
| `Q` | Easy |
| `W` | No Fail |
| `E` | Half Time |
| `A` | Hard Rock |
| `S` | Sudden Death |
| `S` дважды | Perfect |
| `D` | Double Time |
| `D` дважды | Nightcore |
| `F` | Hidden (osu!, osu!taiko, osu!catch) или Fade In (osu!mania) |
| `F` дважды | Hidden (osu!mania) |
| `G` | Flashlight |
| `Z` | Relax (osu!, osu!taiko, osu!catch) или переключение между модами xK (osu!mania) |
| `X` | Auto Pilot (osu!) или Random (osu!mania) |
| `C` | Spun Out (osu!) |
| `V` | Auto |
| `V` дважды | Cinema |
| `B` | Score V2 |
| от `3` до `9` | от 3K до 9K (osu!mania) |

## Во время игры

| Горячая клавиша | Действие |
| :-- | :-- |
| `+` или `-` | Поменять локальный оффсет (зажав `Alt`, можно увеличить точность). |
| `Esc` | Пауза, либо выход в меню выбора песни, если вы наблюдаете за другим игроком или смотрите реплей с модами Auto или Cinema. |
| `Space` | Пропустить вступление карты. |
| `Tab` | Показать/скрыть таблицу рекордов. |
| `Shift` + `Tab` | Показать/скрыть интерфейс. |
| `H` | Показать/скрыть таблицу рекордов и интерфейс реплея (при просмотре реплея или наблюдении за игрой). |
| `F3` или `F4` | Настроить скорость в osu!mania (работает только в самом начале карты). |
| `Ctrl` + `R` или `` ` `` | При удержании нажатия — быстрый перезапуск карты (автоматически пропускает вступление). |

## Режим просмотра реплея

*Примечание: эти клавиши работают только в ветке обновлений cutting edge.*

| Горячая клавиша | Действие |
| :-- | :-- |
| `5` | Поставить реплей на паузу. |
| `6` или `0` | Продолжить реплей, либо поставить обычную скорость воспроизведения. |
| `7` | 2-кратная скорость воспроизведения. |
| `8` | 3-кратная скорость воспроизведения. |
| `9` | 4-кратная скорость воспроизведения. |

## Редактор карт

*Основная статья: [Редактор карт](/wiki/Client/Beatmap_editor)*

### Основные клавиши

Эти сочетания работают на любом экране редактора.

| Горячая клавиша | Действие |
| :-- | :-- |
| `Z` | Перейти к первой ноте или к началу песни. |
| `V` | Перейти к последней ноте или к концу песни. |
| `X` | Воспроизвести песню с начала. |
| `C` или `Пробел` | Пауза/воспроизведение. |
| `Стрелка влево` или `Стрелка вправо` | Перемещение по карте. |
| `Стрелка вверх` или `Стрелка вниз` | Перемещение между тайминг-секциями. |
| `Shift` + `Стрелка влево` или `Shift` + `Стрелка вправо` | Перемещение по карте с 4-кратной скоростью. |
| `Ctrl` + `B` | Поставить закладку на текущей позиции. |
| `Ctrl` + `Shift` + `B` | Удалить закладку с текущей позиции. |
| `Ctrl` + `Стрелки влево/вправо` | Перейти к предыдущей/следующей закладке. |
| `Ctrl` + `L` | Вернуться к последней сохранённой версии карты. Это действие перезагружает файлы `.osu` и `.osb`. |
| `Ctrl` + `Shift` + `L` | Вернуться к последней сохранённой версии карты (полностью). Это действие перезагружает все файлы, используемые в карте. |
| `Ctrl` + `O` | Открыть окно выбора сложности. |
| `Ctrl` + `S` | Сохранить текущее состояние карты. |
| `Ctrl` + `N` | Удалить все объекты на карте. |
| `Ctrl` + `Стрелки вверх/вниз` | Увеличить/уменьшить скорость воспроизведения на 25%. |
| `Ctrl` + `Shift` + `Стрелки вверх/вниз` | Увеличить/уменьшить скорость воспроизведения на 5%. |
| `F1` | Перейти на вкладку [Compose](/wiki/Client/Beatmap_editor/Compose). |
| `F2` | Перейти на вкладку [Design](/wiki/Client/Beatmap_editor/Design). |
| `F3` | Перейти на вкладку [Timing](/wiki/Client/Beatmap_editor/Timing). |
| `F4` | Открыть [настройки карты](/wiki/Client/Beatmap_editor/Song_setup). |
| `F5` | Начать тестирование карты. |
| `F6` | Открыть панель настроки тайминга. |
| `Ctrl` + `P` | Открыть окно настроек тайминга и добавить «красную» тайминг-секцию. |
| `Ctrl` + `Shift` + `P` | Открыть окно настроек тайминга и добавить «зелёную» тайминг-секцию. |
| `Ctrl` + `I` | Удалить текущую тайминг-секцию. Не работает для самой первой «красной» секции. |

#### Управление мышью

| Горячая клавиша | Действие |
| :-- | :-- |
| `Прокрутка колёсика мыши вверх/вниз` | Промотать песню вперед/назад. |
| `Shift` + `Прокрутка колёсика мыши вверх/вниз` | Промотать песню вперед/назад на 4-кратной скорости. |
| `Ctrl` + `Прокрутка колёсика мыши вверх/вниз` | Увеличить/уменьшить шаг временно́й шкалы. |
| `Alt` + `Прокрутка колёсика мыши вверх/вниз` | Если курсор на игровом поле, изменить множитель спейсинга. Если курсор над верхней временно́й шкалой — изменить её масштаб. |
| `Alt` + `Shift` + `Прокрутка колёсика мыши вверх/вниз` | Если курсор на игровом поле, изменить множитель спейсинга (повышенная точность). |

| Горячая клавиша | Действие |
