# ショートカットキーの一覧

osu!にはたくさんのショートカットキーがあります。ゲームプレイ中または任意の画面でアイドル状態のときに使用できます。

*注: このwikiでは、デフォルトのキー設定でのショートカットを記載しています。キー設定は[オプション](/wiki/Client/Options)から変更できます。*

## ゲーム全体

これらのショートカットはどこでも動作します。

| キー | 動作 |
| :-- | :-- |
| `Alt` + `F4` | ゲームクライアントを閉じます (終了確認画面は表示されません)。ゲームプレイ中、エディット中の場合は曲の選択画面に、osu!directを開いている場合はメインメニューに戻ります。 |
| `Alt` + `上/下矢印` | 主音量を調整します。左右の矢印を使うことによって、音楽と効果音の音量を変えることができます。 |
| `Ctrl` + `Alt` + `Shift` + `S` | 現在のスキンをリロードします。ゲームプレイ中または結果画面で使用した場合、曲の選択画面に戻った後に実行されます。 |
| `Ctrl` + `F11` | FPSグラフの表示を切り替えます。`Ctrl`を押すと一時停止します。 |
| `F7` | フレームリミッター制限を切り替えます。 |
| `F8` | チャット画面の表示を切り替えます。 |
| `F9` | 拡張チャット画面の表示を切り替えます。 |
| `F10` | ゲームプレイ中のマウスボタンの無効化を切り替えます。 |
| `F12` | スクリーンショットを撮ります。表示されるダイアログをクリックすると、スクリーンショットのフォルダーが開きます。 |
| `Shift` + `F12` | スクリーンショットを撮り、共有のためにサーバーにアップロードします。ローカルにも保存されます。ブラウザでスクリーンショットが開きます。 |
| `Insert` | [ボスキー](https://ja.wikipedia.org/wiki/%E3%83%9C%E3%82%B9%E3%81%8C%E6%9D%A5%E3%81%9F)です。使用するとタスクバーに最小化され、ゲームがミュートされます。 |
| `Shift` + `F1` | [osu!status](https://twitter.com/osustatus)をブラウザで開きます (接続エラーが表示された場合のみ) 。 |
| `Ctrl` + `O` | オプションを開きます。オフセットウィザードはメインメニューにいないと開けません。 |

## メインメニュー

| キー | 動作 |
| :-- | :-- |
| `Alt` + `Enter` | フルスクリーンモードを切り替えます。 |
| `上/下矢印` | 主音量を調整します。左右の矢印を使うことによって、音楽と効果音の音量を変えることができます。 |
| `Esc` | 戻るか、キャンセルします。 |
| `P`, `E`, `O` | プレイメニューを表示します。 |
| `P`3回 | 曲選択画面に移動します。 |
| `P`2回の後に`M` | [マルチプレイ](/wiki/Client/Interface/Multiplayer)画面に移動します。 |
| `E`2回 | エディットモードに移動します。 |
| `O` | オプションを開きます。 |
| `Q` | 終了確認画面を表示します。 |
| `Esc` | オプションとプレイメニューが閉じられている場合に、終了確認画面を表示します。 |
| `F` | FPSカウンターの表示を切り替えます。 |

### ジュークボックス

| キー | 動作 |
| :-- | :-- |
| `Z`, `F1` | 前の曲を再生します。 |
| `X`, `F2` | 曲を再生します。既に再生されている場合は、最初から再生されます。 |
| `C` | 一時停止をします。 |
| `F4` | 曲を停止します。 |
| `V`, `F5` | 次の曲を再生します。 |
| `J`, `F6` | メニューを開きます。 |
| `R` | ランダムな曲を再生します。 |

## 曲選択画面

*注: マルチプレイでの曲選択画面では、これらの機能が全て動作する訳ではありません。*

| キー | 動作 |
| :-- | :-- |
| `F1` | [Mod選択メニュー](/wiki/Gameplay/Game_modifier)を表示します。 |
| `F2` | ビートマップをランダムに選択します。 |
| `Shift` + `F2` | 1つ前のランダムな譜面を選択します。 |
| `F3` | ビートマップオプションを表示します。 |
| `F5` | ビートマップを再読み込みします (確認画面が表示される可能性があります) 。 |
| `Shift` + `Delete` | ビートマップの削除確認画面を表示します。単一難易度の削除、全ての難易度の削除、曲選択画面に表示されている全てのビートマップの削除から選択できます。 |
| `Ctrl` + `1` | ゲームモードを[osu!](/wiki/Game_mode/osu!)に切り替えます。 |
| `Ctrl` + `2` | ゲームモードを[osu!taiko](/wiki/Game_mode/osu!taiko)に切り替えます。 |
| `Ctrl` + `3` | ゲームモードを[osu!catch](/wiki/Game_mode/osu!catch)に切り替えます。 |
| `Ctrl` + `4` | ゲームモードを[osu!mania](/wiki/Game_mode/osu!mania)に切り替えます。 |
| `Ctrl` + `Enter` | [Auto mod](/wiki/Gameplay/Game_modifier/Auto)でビートマップをプレイします。 |
| `Ctrl` + `Shift` + `Enter` | [Cinema mod](/wiki/Gameplay/Game_modifier/Cinema)でビートマップをプレイします。 |
| `Ctrl` + `A` | Auto modの有無を切り替えます。 |
| `Ctrl` + `F3`, `Ctrl` + `+` | osu!maniaのスクロール速度を上げます (osu!maniaモードの場合) 。 |
| `Ctrl` + `F4`, `Ctrl` + `-` | osu!maniaのスクロール速度を下げます (osu!maniaモードの場合) 。 |
| `Ctrl` + `Backspace` | 検索バーの最後の単語 (スペース区切り) を削除します。 |
| `上/下矢印` | 同じビートマップパック内にある難易度を切り替えます。 |
| `左/右矢印` | ビートマップパックを切り替えます。 |
| `Page Up/Down` | 曲選択画面をスクロールします。 |
| `Enter` | ビートマップを選択している場合、そのビートマップを再生し、コレクションなどのグループを選択している場合は、そのセクションを開閉します。 |
| `Shift` + `左/右矢印` | 次/前のグループを展開します。 |
| `Shift` + `Enter` | 選択したグループを開閉します。 |

## ゲームプレイ中

| キー | 動作 |
| :-- | :-- |
| `+`, `-` | ローカルオフセットを変更します(細かく調整するには`Alt`をホールドします)。 |
| `Esc` | ゲームを一時停止します。Auto、Cinemaが有効になっている場合やリプレイを再生している場合は曲選択画面に戻ります。 |
| `Space` | イントロを飛ばします。 |
| `Tab` | スコアボードの表示を切り替えます。 |
| `Shift` + `Tab` | ゲーム内UIの表示を切り替えます。 |
| `H` | リプレイを見たり観戦したりするときにスコアボードとリプレイインターフェイスの非表示を切り替えます。 |
| `F3`, `F4` | osu!maniaでのスクロール速度を変更します (ゲーム開始時にのみ調整できます) 。 |
| `Ctrl` + `R`, `` ` `` | 短い長押しが必要です。現在の譜面をリトライします (リトライ時にイントロをスキップします) 。 |

## リプレイモード

*注: これらの機能を利用するには、バージョンをcuttingedgeにする必要があります。*

| キー | 動作 |
| :-- | :-- |
| `5` | リプレイを一時停止します。 |
| `6`か`0` | リプレイを再開または再生速度を1倍にします。 |
| `7` | 再生速度を2倍にします。 |
| `8` | 再生速度を3倍にします。 |
| `9` | 再生速度を4倍にします。 |

## ビートマップエディター

*メインページ: [ビートマップエディター](/wiki/Client/Beatmap_editor)*

### 全般

これらはビートマップエディターのどこでも機能します。

| キー | 動作 |
| :-- | :-- |
| `Z` | 再生位置を最初のオブジェクトまたは曲の最初に移動します。 |
| `V` | 再生位置を最後のオブジェクトまたは曲の最後に移動します。 |
| `X` | 最初から再生します。 |
| `C`か`Space` | 一時停止または再開します。 |
| `左矢印`か`右矢印` | 曲をスキップします。 |
| `上矢印`か`下矢印` | タイミングポイント間を移動します。 |
| `Shift` + `左矢印`か`Shift` + `右矢印` | 4倍の速度で曲をスキップします。 |
| `Ctrl` + `B` | 現在のタイムスタンプでブックマークを追加します。 |
| `Ctrl` + `Shift` + `B` | 現在のタイムスタンプのブックマークを削除します。 |
| `Ctrl` + `左/右矢印` | 前/次のブックマークにジャンプします。 |
| `Ctrl` + `L` | 最後に保存した状態 (部分的) に戻します。これにより、`.osu`と`.osb`ファイルが再読み込みされます。 |
| `Ctrl` + `Shift` + `L` | 最後に保存した状態 (完全) に戻します。すべてのファイルが再読み込みされます。 |
| `Ctrl` + `O` | 難易度選択ウィンドウを開きます。 |
| `Ctrl` + `S` | ビートマップを保存します。 |
| `Ctrl` + `N` | ヒットオブジェクトをすべて削除する確認ウィンドウを表示します。 |
| `Ctrl` + `上/下矢印` | 再生速度を25%増減します。 |
| `Ctrl` + `Shift` + `上/下矢印` | 再生速度を5%増減させます。 |
| `F1` | [compose](/wiki/Client/Beatmap_editor/Compose)タブに切り替えます。 |
| `F2` | [design](/wiki/Client/Beatmap_editor/Design)タブに切り替えます。 |
| `F3` | [timing](/wiki/Client/Beatmap_editor/Timing)タブに切り替えます。 |
| `F4` | [song setup](/wiki/Client/Beatmap_editor/Song_setup)メニューを表示します。 |
| `F5` | テストモードに入ります。 |
| `F6` | タイミングとコントロールポイントのメニューを開きます。 |
| `Ctrl` + `P` | 継承されていないタイミングポイントを現在の場所に追加します。タイミングとコントロールポイントのメニューも開きます。 |
| `Ctrl` + `Shift` + `P` | 継承されたタイミングポイントを現在の場所に追加します。タイミングとコントロールポイントのメニューも開きます。 |
| `Ctrl` + `I` | 現在のタイミングポイントを削除します。これは、継承されていない最初のタイミングポイントでは機能しません。 |

#### マウス

| キー | 動作 |
| :-- | :-- |
| `マウスホイール上下` | 再生位置を前後に移動します。 |
| `Shift` + `マウスホイール上下` | 再生位置を4倍の速度で前後に移動します。 |
| `Ctrl` + `マウスホイール上下` | ビートスナップ間隔を調整します。 |
| `Alt` + `マウスホイール上下` | ノート間の距離を調整します。タイムライン上にカーソルを合わせるとタイムラインのズームを調整します。 |
| `Alt` + `Shift` + `マウスホイール上下` | ノート間の距離を細かく調整します。 |

#### オブジェクト操作

| キー | 動作 |
| :-- | :-- |
| `Q` | 新しいコンボを切り替えます。 |
| `Ctrl` + `,` | 選択範囲を反時計回りに90度回転させます。 |
| `Ctrl` + `.` | 選択範囲を時計回りに90度回転させます。 |

| `Ctrl` + `G` | 選択範囲のオブジェクトの順番を逆にします。つまり、最初のオブジェクトは最後のオブジェクトに移動するというようにします。 |

| `Ctrl` + `上/下矢印` | 選択したヒットオブジェクトを1/4の間隔で上下に移動します (osu!mania) 。 |
