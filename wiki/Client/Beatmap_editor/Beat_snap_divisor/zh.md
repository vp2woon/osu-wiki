# 音符时值 (Beat Snap Divisor)

![](img/beat_snap_divisor.png "谱面编辑器中的节拍细分设置")

**音符时值 (Beat Snap Devisor)** 是一项编辑器设置，用来决定编辑器在[节拍固定](/wiki/Client/Beatmap_editor/Beat_snap)过程中量化[节拍](/wiki/Music_theory/Beat)的行为。你可以在编辑器界面的右上部找到此设置。

音符时值与[时间轴](/wiki/Client/Beatmap_editor/Timelines)上[打击物件](/wiki/Gameplay/Hit_object)的密度相对应。时值以一整拍应被分成的数目的倒数表示；音符时值越小，同一段时间内就可以放置更多的物件，反之亦然。

## 支持的时值

从 1/1 到 1/16，谱面编辑器支持 11 种不同的音符时值设置。

| 时值 | 节拍线颜色 | 显示 |
| :-- | :-- | :-- |
| 1/1 | 白色 | ![](img/1_1_snap_divisor.jpg "物件时间轴上的 1/1 细分时值") |
| 1/2 | 红色 | ![](img/1_2_snap_divisor.jpg "物件时间轴上的 1/2 细分时值") |
| 1/3 | 紫色 | ![](img/1_3_snap_divisor.jpg "物件时间轴上的 1/3 细分时值") |
| 1/4 | 蓝色 | ![](img/1_4_snap_divisor.jpg "物件时间轴上的 1/4 细分时值") |
| 1/5 | 黄色 | ![](img/1_5_snap_divisor.png "物件时间轴上的 1/5 细分时值") |
| 1/6 | 紫色 | ![](img/1_6_snap_divisor.jpg "物件时间轴上的 1/6 细分时值") |
| 1/7 | 黄色 | ![](img/1_7_snap_divisor.png "物件时间轴上的 1/7 细分时值") |
| 1/8 | 黄色 | ![](img/1_8_snap_divisor.jpg "物件时间轴上的 1/8 细分时值") |
| 1/9 | 黄色 | ![](img/1_9_snap_divisor.png "物件时间轴上的 1/9 细分时值") |
| 1/12 | 灰色 | ![](img/1_12_snap_divisor.png "物件时间轴上的 1/12 细分时值") |
| 1/16 | 灰色 | ![](img/1_16_snap_divisor.png "物件时间轴上的 1/16 细分时值") |

由于多数含有较密节奏的歌曲使用，1/1（整拍）、1/2（半拍）和 1/4（四分之一拍）时值是最普遍的。给华尔兹歌曲作图时，经常会用到 1/3（三分音符）和 1/6（六分音符）这样的时值，其中一拍被三等分或六等分。

剩下的音符时值设置不常见，应该谨慎使用：除非歌曲或其中一段特别使用了特殊的节拍长度，否则，如果谱面内出现了罕见的时值（比如 1/5 和 1/16），那么这一般表明谱面的[时间轴](/wiki/Beatmapping/Timing)有问题。但高频折返滑条 (buzz slider) 常常用到的 1/16 时值除外。
