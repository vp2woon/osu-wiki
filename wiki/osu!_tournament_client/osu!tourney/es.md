# osu!tourney

*Véase también: [Cliente para torneos de osu!](/wiki/osu!_tournament_client)*

**osu!tourney** es el cliente multiespectador para la versión estable de osu!, que se usa para ver y transmitir a todos los jugadores en una sala multijugador a la vez.

Los usuarios que encuentren **problemas** con el cliente pueden leer la [guía de solución de problemas](Troubleshooting) o enviar un correo electrónico a [tournaments@ppy.sh](mailto:tournaments@ppy.sh).
