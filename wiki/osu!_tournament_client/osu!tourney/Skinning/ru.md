# Скиннинг osu!tourney

![Оформление клиента можно настроить](img/Osutourneycustom.png)

Клиент можно различным образом оформить под отдельные турниры. Для этого в папке с инсталляцией osu!tourney создайте папку по пути `/Skins/User/tournament`. В неё нужно будет добавить элементы скина с расширениями `.jpg` или `.png`.

Настроить можно следующее:

- `background` - фоновое изображение. Стандартный фон можно скачать [отсюда](https://s.ppy.sh/images/tournament/default.png).
- `background-win1` (необязательно) - фон, который постепенно появится поверх результатов при победе синей команды.
- `background-win2` (необязательно) - фон, который постепенно появится поверх результатов при победе красной команды.
- `tourney-title` (необязательно) - изображение, которое показывается в нижней части клиента поверх его фона.
  - Здесь, например, можно разместить логотип турнира.

![К названиям команд можно добавить значки](img/Osutourneyicons.png)

Около названий команд можно разместить значки из папки `/Skins/User/tournament/icons`. В качестве значков можно использовать, например, аватарки команд или флаги стран.

Названия значков должны совпадать с названиями команд. Например, если комната названа `Test Tourney: (Team 1) vs (Team 2)`, то значки должны называться `Team 1.jpg` и `Team 2.jpg`. Поддерживаемые форматы файлов — `.jpg` и `.png`, оптимальное разрешение — `50x50px`.

Шаблон скина для оформления клиента можно [скачать отсюда](https://s.ppy.sh/images/tournament/template.zip).
