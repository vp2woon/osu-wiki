# Blanket

::: Infobox
![](img/blanket.png "Sebuah blanket hit object 2 dengan menggunakan sliderbody dari hit object 1")
:::

**Blanket** adalah efek yang dibuat saat [sliderbody](/wiki/Gameplay/Hit_object/Slider/Sliderbody) melingkari bentuk [hit object](/wiki/Gameplay/Hit_object) lain. Blanketing dapat dilakukan dengan berbagai cara agar sesuai dengan gaya visual [beatmap](/wiki/Beatmap). Panduan untuk membuatnya dapat ditemukan di [artikel teknik mapping](/wiki/Beatmapping/Mapping_techniques/Making_good_sliders#beat-blankets).
