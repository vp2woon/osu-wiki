# Blanket

::: Infobox
![](img/blanket.png "A blanket of hit object 2 using the sliderbody of hit object 1")
:::

A **blanket** is the effect created when a [sliderbody](/wiki/Gameplay/Hit_object/Slider/Sliderbody) wraps around the shape of another [hit object](/wiki/Gameplay/Hit_object). Blanketing can be done in many ways to fit the visual style of a [beatmap](/wiki/Beatmap). A guide to making them can be found in the [mapping techniques article](/wiki/Beatmapping/Mapping_techniques/Making_good_sliders#beat-blankets).
