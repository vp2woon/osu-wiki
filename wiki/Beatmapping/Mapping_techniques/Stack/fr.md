# Stack

![](img/stack.jpg "Un exemple de deux cercles empilés sur un sliderhead.")

Un **stack** est un ensemble d'[objets](/wiki/Gameplay/Hit_object) qui sont en [overlap](/wiki/Beatmapping/Mapping_techniques/Overlap) les uns sur les autres dans le champ de jeu. Les objets les plus courants qui sont stackés sont les [cercles](/wiki/Gameplay/Hit_object/Hit_circle).

Les stacks sont automatiquement créés lorsque les objets sont proches de se chevaucher parfaitement. Le temps entre les objets pour que cela se produise est déterminé par l'[indulgence du stacking](/wiki/Beatmap/Stack_leniency) de la beatmap.
