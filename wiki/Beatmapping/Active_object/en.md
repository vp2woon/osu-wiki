# Active object

The **active object** is a term used in [osu!catch](/wiki/Game_mode/osu!catch) for all objects which have to be caught in order to maintain [combo](/wiki/Beatmapping/Combo), this includes [fruits](/wiki/Gameplay/Hit_object/Fruit) and [drops](/wiki/Gameplay/Hit_object/Juice_stream#drop).
