# Вимоги для отримання медалей Mod Introduction

| Назва медалі | Вимога |
| :-- | :-- |
| Finality | Пройти кваліфіковану, ранкнуту, або схвалену бітмапу з [Sudden Death](/wiki/Gameplay/Game_modifier/Sudden_Death) модом. |
| Perfectionist | Пройти кваліфіковану, ранкнуту, або схвалену бітмапу з [Perfect](/wiki/Gameplay/Game_modifier/Perfect) модом. |
| Rock Around The Clock | Пройти кваліфіковану, ранкнуту, або схвалену бітмапу з [Hard Rock](/wiki/Gameplay/Game_modifier/Hard_Rock) модом. |
| Time And A Half | Пройти кваліфіковану, ранкнуту, або схвалену бітмапу з [Double Time](/wiki/Gameplay/Game_modifier/Double_Time) модом. |
| Sweet Rave Party | Пройти кваліфіковану, ранкнуту, або схвалену бітмапу з [Nightcore](/wiki/Gameplay/Game_modifier/Nightcore) модом. |
| Blindsight | Пройти кваліфіковану, ранкнуту, або схвалену бітмапу з [Hidden](/wiki/Gameplay/Game_modifier/Hidden) модом. |
| Are You Afraid Of The Dark? | Пройти кваліфіковану, ранкнуту, або схвалену бітмапу з [Flashlight](/wiki/Gameplay/Game_modifier/Flashlight) модом. |
| Dial It Right Back | Пройти кваліфіковану, ранкнуту, або схвалену бітмапу з [Easy](/wiki/Gameplay/Game_modifier/Easy) модом. |
| Risk Averse | Пройти кваліфіковану, ранкнуту, або схвалену бітмапу з [NoFail](/wiki/Gameplay/Game_modifier/No_Fail) модом. |
| Slowboat | Пройти кваліфіковану, ранкнуту, або схвалену бітмапу з [Half Time](/wiki/Gameplay/Game_modifier/Half_Time) модом. |
| Burned Out | Пройти кваліфіковану, ранкнуту, або схвалену бітмапу з [Spun Out](/wiki/Gameplay/Game_modifier/Spun_Out) модом. Цю медаль можна отримати тільки в режимі гри [osu!](/wiki/Game_mode/osu!), через те, що моду `Spun Out` немає в інших режимах гри. |
