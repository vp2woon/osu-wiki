| Nombre de la medalla | Requisito |
| :-- | :-- |
| Finality | Pasa un mapa calificado, clasificado o aprobado con el mod [Sudden Death](/wiki/Gameplay/Game_modifier/Sudden_Death) habilitado. |
| Perfectionist | Pasa un mapa calificado, clasificado o aprobado con el mod [Perfect](/wiki/Gameplay/Game_modifier/Perfect) habilitado. |
| Rock Around The Clock | Pasa un mapa calificado, clasificado o aprobado con el mod [Hard Rock](/wiki/Gameplay/Game_modifier/Hard_Rock) habilitado. |
| Time And A Half | Pasa un mapa calificado, clasificado o aprobado con el mod [Double Time](/wiki/Gameplay/Game_modifier/Double_Time) habilitado. |
| Sweet Rave Party | Pasa un mapa calificado, clasificado o aprobado con el mod [Nightcore](/wiki/Gameplay/Game_modifier/Nightcore) habilitado. |
| Blindsight | Pasa un mapa calificado, clasificado o aprobado con el mod [Hidden](/wiki/Gameplay/Game_modifier/Hidden) habilitado. |
| Are You Afraid Of The Dark? | Pasa un mapa calificado, clasificado o aprobado con el mod [Flashlight](/wiki/Gameplay/Game_modifier/Flashlight) habilitado. |
| Dial It Right Back | Pasa un mapa calificado, clasificado o aprobado con el mod [Easy](/wiki/Gameplay/Game_modifier/Easy) habilitado. |
| Risk Averse | Pasa un mapa calificado, clasificado o aprobado con el mod [NoFail](/wiki/Gameplay/Game_modifier/No_Fail) habilitado. |
| Slowboat | Pasa un mapa calificado, clasificado o aprobado con el mod [Half Time](/wiki/Gameplay/Game_modifier/Half_Time) habilitado. |
| Burned Out | Pasa un mapa calificado, clasificado o aprobado con el mod [Spun Out](/wiki/Gameplay/Game_modifier/Spun_Out) habilitado. Esta medalla solo se puede conseguir en el modo de juego [osu!](/wiki/Game_mode/osu!) ya que el mod `Spun Out` no est√° presente en otros modos de juego. |
