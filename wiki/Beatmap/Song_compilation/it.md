# Compilation di brani

Una **compilation di brani** è una collezione di tre o più brani, interi o parziali, che di solito hanno un genere, artista o altro tema in comune.

Le [beatmap](/wiki/Beatmap) delle compilation di brani usano un singolo file audio contenente l'intera collezione di brani, che di solito include solo un ritornello o verso di ciascuno. Questo è da non confondere con le beatmap che includono più file di brani nella propria cartella.
