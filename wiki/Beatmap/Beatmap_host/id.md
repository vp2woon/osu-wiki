# Beatmap host

**Beatmap host** adalah manajer dan pengunggah dari sebuah [beatmap](/wiki/Beatmap). Mereka kebanyakan membuat mapsets difficulties dan mengkordinasi pembuatan sebuah [guest difficulties](/wiki/Beatmap/Guest_difficulty). Seorang beatmap host biasanya bertanggung jawab untuk mencari [mods](/wiki/Modding) dan menghubungi [Beatmap Nominators](/wiki/People/Beatmap_Nominators) untuk meneruskan beatmap mereka ke tahap [prosedur beatmap ranking](/wiki/Beatmap_ranking_procedure).
