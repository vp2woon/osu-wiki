# Mod request

**Mod request(モッドリクエスト)** とはクリエイターが[ビートマップ](/wiki/Beatmap)をより良くするために他のマッパーに依頼することである。Normal modや[Modding queueフォーラム](https://osu.ppy.sh/community/forums/60)での[mod for mod](/wiki/Modding/Mod_for_mod)など様々な方法で行われます。リクエストが承認されたら[Modding](/wiki/Modding)が始まり、ビートマップに添削や改善策が投稿されます。
