# Solicitud de mod

Una **solicitud de mod** es un término que se usa cuando un creador necesita ayuda y quiere mejorar su [beatmap](/wiki/Beatmap). Esto se puede hacer de muchas maneras, pero la más común es solicitar un mod normal o [mod por mod](/wiki/Modding/Mod_for_mod) en una de las [colas de modding del foro](https://osu.ppy.sh/community/forums/60). Una vez que se ha aceptado la solicitud de mod, se inicia el proceso de [modding](/wiki/Modding) y se revisará el beatmap para detectar posibles errores y mejoras.
