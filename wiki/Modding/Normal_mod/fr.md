# Normal mod

*Voir aussi : [NM (désambiguïsation)](/wiki/Disambiguation/NM).*

Un **normal mod (*NM*)** est un [modding](/wiki/Modding) qui est fourni sans nécessiter de travail de la part du créateur demandeur. Les normal mod sont généralement réalisés pour obtenir un deuxième avis sur une [beatmap](/wiki/Beatmap) afin de l'améliorer. De nombreux moddeurs utilisent le terme "NM" pour le distinguer des échanges [mod for mod](/wiki/Modding/Mod_for_mod).

Les normal mods sont le plus souvent demandés sur les forums dans les [files d'attente de modding](https://osu.ppy.sh/community/forums/60), où les moddeurs peuvent personnaliser le type de beatmaps qu'ils souhaitent voir, et les mappeurs peuvent trouver des personnes prêtes à les aider.
