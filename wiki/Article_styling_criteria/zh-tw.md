# 條目風格準則

*關於新聞稿，請見：[新聞風格準則](/wiki/News_styling_criteria)*

條目風格準則 (ASC) 作為 osu! wiki 的強制性風格標準，以保持所有條目在清晰度、格式，及樣式的一致性，並盡力實現文法及資訊的正確性。

這些條目主要為審查時的輔助工具，代表了 osu!wiki 貢獻者多年來所形成的共識。wiki 的審查流程不是單打獨鬥，是多人合作的結果，所以您不需要將整個 ASC 完全閱覽或熟記。如果您也想要貢獻，請見 [貢獻指南](/wiki/osu!_wiki/Contribution_guide)。

若要對條目風格準則提出修改建議，[請到 Github 上發起 issue](https://github.com/ppy/osu-wiki/issues/new)。

## 準則

*提醒：下列條目使用 [RFC 2119](https://tools.ietf.org/html/rfc2119) 來描述必要等級。*

條目風格準則分為兩篇：

- [格式](Formatting)：包含 Markdown 及其他格式準則
- [編寫](Writing)：包含寫作手法及文法準則
