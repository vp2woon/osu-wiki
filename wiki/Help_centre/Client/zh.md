# 客户端

*主页面：[帮助中心](/wiki/Help_centre)*

游戏客户端遇到问题了吗？你可以在本文中找一找，看看有没有我们的用户最常遇到的问题。

## Bug 和崩溃 {id=crash}

### 我修改了设置，现在 osu! 打不开或直接卡住了！ {id=incorrect-settings}

**大多数情况下，将 osu! 设置重置为默认值可以解决此问题。**

按照以下步骤做来恢复 osu! 的默认设置：

1. 按住 `Shift` 按键启动 osu!。
2. 持续按住 `Shift`，直到 osu! recovery 对话框出现。
3. 当 `osu! configuration` 对话框出现时，点击 `reset settings`。
4. 如果问题没有解决，请按上述步骤再次打开配置窗口，并点击 `repair osu!`。

### 点击链接或者点击下载多人游戏谱面后，osu! 就卡住了！ {id=broken-links}

**请确保你没有使用 Windows 兼容模式运行 osu!，且在系统设置中指定了一个默认浏览器。**

这个问题有时是因为兼容模式与游戏冲突，也有可能只是因为 osu! 无法使用默认浏览器打开链接。

#### 你的游戏正在使用兼容模式运行吗？ {id=windows-compatibility}

在 Windows 下，若要检查你的游戏是否使用兼容模式运行，请按以下步骤做：

1. 打开 osu! 游戏文件夹，找到 `osu!.exe`。
2. 右键点击，然后在菜单中选择`属性`。
3. 转到`兼容性`标签。
4. 在名为`兼容模式`的区域下，找到`以兼容模式运行这个程序`选项。
5. 如果这个选项打钩了，点一下来取消。
6. 如果游戏仍在运行，重启游戏。

#### 你设置默认浏览器了吗？ {id=default-browser}

**绝大多数情况下，Windows 安装时就会自动设置默认浏览器。**

按照以下步骤检查（Windows 8 及更高版本）：

1. 打开开始菜单。
2. 在开始菜单的搜索框里输入`默认设置`，然后点击出现的设置。
3. 向下滚动，找到`网页浏览器`，检查有没有设置为已安装的浏览器。

### 我的游戏无法完成更新！ {id=cannot-update}

**这通常是因为你的电脑所致，极少数情况下，也可能是因为游戏的新版本有问题。**

osu! 通常会自动保持版本最新、无需干涉，但有时可能会出现问题。

我们建议你先重启一下你的电脑。对于电脑来说，重启能解决的问题会超出你的预期，对于 osu! 也不例外。

确保游戏设置中的“更新通道”使用的是`稳定版`。

你还可以尝试强制升级。

#### 更改更新版本 {id=release-stream}

**可在游戏内直接更改相关设置。**

1. 打开 osu!。
2. 在主菜单点击 `Options`，或者按下 `Ctrl` + `O`。
3. 搜索 `release`，可以直接看到更新版本设置。
4. 确保你选择的是`正式版（每月更新）`，这样才可以收到最稳定可靠的游戏版本。

#### 强制更新 osu! {id=force-update}

**你可以在设置里强制更新 osu!。**

1. 打开 osu!。
2. 在主菜单点击 `Options`，或者按下 `Ctrl` + `O`。
3. 搜索 `update`，可以直接看到强制更新设置。
4. 点击`启动 osu! 更新程序`。

你还可以在 osu! 游戏文件夹下创建一个 `help.txt` 来进行强制更新。文件不需要包含任何内容，只要文件存在，游戏下一次启动的时候就会强制进行更新了。

### 我在打开游戏时，看到了关于“字体样式”的错误！ {id=no-default-fonts}

**你删除了 Windows 安装时自带的基础字体，或是字体已经丢失。同时，你可能也注意到了其他问题。**

如果你在打开 osu! 或者 osu! 更新程序时看到这种报错，就意味着 Windows 安装时自带的重要系统字体被删除或丢失了。整个 Windows 系统都会使用这些字体，所以你也会遇到其他各种问题，例如字体错误、文本无法显示、其他程序崩溃等等。

以下是 Windows 各版本和互联网上都普遍使用的字体，可能对你有帮助：

- [Microsoft core web fonts](https://web.archive.org/web/20020124085641/http://www.microsoft.com/typography/fontpack/default.htm)（尤其是 Arial、Times New Roman、Trebuchet MS 和 Verdana）
- [Tahoma](https://freefontsfamily.com/tahoma-font-free)
- [Windows Live Essentials](https://support.microsoft.com/en-us/help/2434419/windows-live-essentials-2011) (Segoe UI)
- [微軟正黑體 (Microsoft JhengHei)](https://microsoft.com/en-us/download/details.aspx?&id=12072)（繁体中文 UI 字体）

在 Microsoft Answers 上查看[这个主题](https://answers.microsoft.com/en-us/windows/forum/windows_vista-windows_programs/font-tahoma-does-not-support-style-regular/80ad7a97-230f-41d4-9101-107a0bfa986a)来获取一些具体建议，以及找回字体的其他方法。

### 我的谱面丢了一部分！ {id=missing-beatmaps}

**请确保你没有使用右上角的分组设置为谱面分组（也就是“分组”选项应设置为“不分组”）！**

如果你刚下载了新谱面，请在选歌页面按下 `F5` 来更新谱面列表。

你也要确认设置中是否限制了谱面显示。

请按照以下步骤检查显示的谱面：

1. 打开 osu!。
2. 在主界面点击 `Options`，或者按下 `Ctrl` + `O`。
3. 在搜索框输入 `song select`，可以直接看到相关设置。
4. 请确保`最低显示星数`为 0，`最高显示星数`为 10+。

![](img/difficulty-display.png "谱面显示设置的正确值")

如果上述步骤无一生效，作为最后的解决方案，你可以尝试重建谱面数据库。**请注意此操作会导致所有谱面被标记为"未玩过"，所以将无法再依据游玩状态和日期进行搜索或分组。**

若要安全重建你的谱面数据库，遵循以下步骤：

1. 打开 osu!。
2. 在主界面点击 `Options`，或者直接按下 `Ctrl` + `O`。
3. 点击`打开 osu! 文件夹`.
4. 关闭 osu!。
5. 在新打开的文件夹里，找到 `osu!.db`。
6. 右键点击，然后点击`重命名`
7. 你可以随意修改名字，只要不是 osu! 就可以。改完后按下回车。
8. 重新打开 osu!。

*提示：2021-05-19更新的 [Stable 20210519.3](https://osu.ppy.sh/home/changelog/stable40/20210519.3) 解决了这个问题。如果问题对你依然存在，请[让我们知道](https://github.com/ppy/osu-stable-issues/issues)。*

### 我的选歌界面一直在滚动！ {id=songs-list-scrolling}

**这通常是因为你的输入设备出了问题。尝试拔掉所有的手柄和控制器。**

也有可能是因为你正在使用会改变或模拟按键绑定的软件，例如 *Xpadder* 或 *X-Mouse Button Control*。如果你玩其他游戏时用到这些程序，请在玩 osu! 时将其关闭。

因为数字小键盘可以用来滚动选歌界面，所以问题也有可能出自于它。按下 `数字锁定 (NumLock)`，然后按下小键盘上的 `9`、`8`、`3`、`2` 即可解决问题。

你还可以检查外设上是否有卡住或损坏的按键。

### 我下载了一个谱面包，但是 osu! 一直尝试“修复”文件！ {id=beatmap-pack-extraction}

**你需要把压缩包解压到歌曲文件夹。**

大多数谱面包的格式为 `.rar`，这意味着它们是包含很多谱面的压缩文件。你需要使用你喜欢的工具(我们推荐 [7-Zip](https://7-zip.org)) 来先把压缩文件解压到你的 `Songs` 文件夹。

在所有的 `.osz` 文件都被解压到 `Songs` 文件夹之后，在选歌界面按下 `F5` 即可刷新歌曲缓存，将新谱面载入游戏。

## 游玩 {id=gameplay}

### 我的连击数、分数和准度消失了！ {id=missing-interface}

**游玩时按下 `Shift` + `Tab` 会显示或隐藏这些元素，再按一次就可以重新看到了。**

如果你更改过计分板的按键绑定，那么请在`设置` -> `按键设置` -> `In-Game` -> `Toggle Scoreboard`当中找到你绑定的按键。你也可以把 `Tab` 更改成其他按键（比如，绑定成 `Shift` + `V`）。

### 我的本地成绩消失了！ {id=no-scores}

多种原因都可能导致此事发生，请继续阅读！

#### 你最近重装游戏了吗？ {id=no-scores-reinstalling}

**重新安装游戏客户端时会自动清除所有本地成绩。**

很遗憾，这些成绩已无法找回了。

但是，你仍然可以找到已提交上线的成绩。只要重新下载你玩过的谱面，成绩就会重新出现。

小贴士：如果你是 osu! 支持者，你可以在 osu!direct 面板里使用 `Ranked（玩过）`来筛选你玩过的谱面。你也可以在[谱面列表页](https://osu.ppy.sh/beatmapsets?played=played&s=ranked)找到你以往留下过成绩的谱面。

#### 你选择了正确的游戏模式吗？ {id=no-scores-game-mode}

**选择其他游戏模式（osu!taiko、osu!catch、osu!mania）会自动隐藏其他模式的谱面成绩，只显示当前模式的成绩。**

游玩其他模式的谱面之后，默认的模式会被自动切换，因此成绩常常被隐藏而导致用户一头雾水。切换回正确的游戏模式即可。

你可以点击选歌界面左下方的 `Mode` 按钮，选择待寻找成绩所在的正确模式。

#### 你最近下载了大量谱面吗？（即谱面包） {id=no-scores-many-maps}

**成绩会在谱面导入后自动从服务器下载，若一次性导入大量谱面，下载成绩可能需要一段时间。**

先玩会游戏，或者做点别的事情吧。游戏会在后台自动下载你的成绩。

### 我光标后面的这个像烟雾的线条或轨迹是什么？ {id=smoke}

**这是一个名为 *拉烟 (Smoke)* 的特殊功能，可以用来在游戏区域里画画。所有旁观游戏的玩家都可以看到你的画作。**

默认情况下此功能绑定 `C` 键，按住此按键就可以使用。你可以在`按键设置`的 `osu!` 这一部分更改按键绑定。

### 如何保存我刚获得成绩的回放？ {id=save-replay}

**点击本地排行榜中的分数进入结算屏幕，然后按下 `F2` 即可。**

你漂亮的新成绩会以 `.osr` 的格式被保存在 osu! 默认安装路径的 /Replays/ 文件夹中。

在你玩完一首歌之后，osu! 还会自动把所有回放保存到游戏路径下的 `/Data/r/` 隐藏文件夹当中。

请知悉，如果成绩本身不附带回放，按下 F2 也不能恢复它。

### osu! 提醒我：我的游戏版本过低！ {id=old-client}

**是时候更新了！不能在过旧的游戏版本内提交新成绩，所以你需要点击 `Options` -> `常规` -> `启动 osu! 更新程序`来强制更新游戏。**

如果此举无效，你可以关闭 osu!，然后按住 `Shift` 并重新打开 osu!。如此做你会看到几个更新和修复选项，其中就有更新游戏的按钮。

### 我的成绩没有被提交！ {id=no-submission}

嘛，这件事情可能由多种原因引起，请逐个检查：

#### 游玩的时候联网了吗？ {id=no-submission-no-connection}

**如果你在一局游戏已经结束时未联网，则成绩可能无法提交。**

网络不稳定确实很令人抓狂，但只要游戏仍在运行，客户端就会努力尝试提交你的成绩。

#### 本地防火墙或杀毒软件允许 osu! 联网吗？ {id=no-submission-firewall}

**部分杀毒软件或防火墙会阻止 osu! 访问网络，进而阻止成绩的提交。**

检查你的杀毒软件设置，确保允许游戏路径下的 `osu!.exe` 联网。

#### 后台是否有太多程序？ {id=no-submission-software}

**部分程序可能会影响成绩的提交。**

如果你的网络没有问题，而 osu! 仍然无法提交你的成绩，那么请在游玩之前关闭任何多余的程序。

#### 你的谱面状态正确吗？ {id=no-submission-beatmap-status}

**部分情况下，谱面状态会出现异常，这会导致你的成绩无法提交。**

在选歌界面检查屏幕左上角的小图标（例如，一张上架 (Ranked) 谱面应该会显示向上的蓝色双箭头）。如果你的谱面没有任何图标，或图标不正确，则可能无法上传成绩。

![](img/correct-map-status.png "正常情况下是这个样子的")

游玩时选择`全球排行榜`有助于避免成绩无法提交的情况发生。如果你的大部分，甚至全部谱面状态都不正常，请参阅[我丢失了一部分谱面！](/wiki/Help_centre/Client#missing-beatmaps)来重建你的谱面数据库。重建后，可能需要一段时间，谱面状态才会恢复正常。

#### 成绩服务器在正常运行吗？ {id=no-submission-servers}

**答案几乎一定是肯定的，但是你可以检查 [osu! 服务器状态页](https://status.ppy.sh) 或者 [@osustatus Twitter](https://twitter.com/osustatus) 来得知是不是我们的锅。**

如果服务器目前不可用，那么在我们修复问题之前，请不要关闭 osu!。osu! 会在至多一个小时内一直尝试提交成绩。

### 你们可以帮我上传成绩吗？ {id=upload-replay}

**很遗憾，我们不能。**

如果成绩因各种问题提交失败，你只能希望再次展现技能并获得那样的分数。抱歉！

### 物件好像和音乐不同步！ {id=offsync-notes}

**这可能是因为你电脑上的音频软件导致了问题，例如 *Razer Surround Audio*。也有可能是因为你的`全局偏移量`设置有误。**

尝试关闭电脑上会处理音频信号的软件。如果没有效果，你可以尝试更改`全局偏移量`。

每一套硬件都会有独特的`全局偏移量`值，这决定了你听到音乐的时机会早一点还是晚一点。

默认值适用于大多数人，但是如果不适合你，你可以跟随下列步骤来进行更改：

1. 打开 osu!。
2. 在主界面点击 `Options`，或按下 `Crtl` + `O`。
3. 在搜索框输入 `offset`，这样就可以看到全局偏移量设置。
4. 使用滑块来更改它。你也可以尝试使用[偏移量设置向导](/wiki/Client/Options/Offset_Wizard)来帮助你调整。

如果还是没有用，你可以尝试更新声卡或主板音频驱动。

### 我的游戏看起来好像有问题，出现了撕裂画面！ {id=screen-tearing}

**这是由名为[画面撕裂](https://baike.baidu.com/item/画面撕裂/15626593)的现象造成的，把`帧率限制`设置更改为`垂直同步`即可解决问题。**

这种现象并非软件或硬件出现问题（虽然看起来有点诡异），单纯是因为游戏的刷新率与你的显示器不相同。

你也可以尝试使用窗口化或全屏无边框模式游玩，这会让 osu! 跟随系统调度垂直同步。

在窗口化或全屏无边框模式下启用垂直同步，会增加*细微的*游戏输入延迟。对大多数人来讲，影响不会很大。

对于较新的 NVidia 显卡，你可以在 NVidia 控制面板中将全局刷新率设置为“快”，这也能解决问题。

## 在线功能 {id=online-features}

### 我无法登录，或是游戏根本无法联网！ {id=cannot-sign-in}

**请确保你电脑上的防火墙或者杀毒软件没有禁止 osu! 的联网权限。**

以下为常见杀毒软件允许程序联网的方式：

#### 在 Windows 防火墙中允许程序联网 {id=firewall-whitelist-windows}

**在大多数 Windows 系统中，Windows 防火墙是默认的防火墙软件。**

若要确保 osu! 可以联网，需要把 osu! 加入 Windows 防火墙的白名单。

1. 打开开始菜单。
2. 在开始菜单的搜索框当中，输入`允许应用`。
3. 你应该会看到名为`允许应用通过 Windows 防火墙`的控制面板项，请点击它。
4. 将会出现一个面板，其中列出了很多程序，每个程序旁边有两个复选框。随便点击一个程序的名字，然后按下 `o` 键。这会让列表滚动到以 "o" 开头的程序那里。
5. 寻找 `osu!` 或 `osu!.exe`。
6. 如果这两个都没有，请点击面板底部的`允许其他应用`，然后搜索你的 osu! 安装路径，在里面选择 `osu!.exe`。
7. 请确保`专用`和`公用`复选框都打上了勾，然后点击确认保存，打开游戏重新尝试。

#### 允许程序通过 AVG 防火墙 {id=firewall-whitelist-avg}

请参阅 AVG 的[官方知识库文章](https://support.avg.com/SupportArticleView?l=en&urlName=AVG-Antivirus-Enhanced-Firewall-Application-Rules&supportType=home)来了解如何操作。

#### 允许程序通过卡巴斯基 (Kaspersky) 互联网安全 {id=firewall-whitelist-kaspersky}

请参阅卡巴斯基的[官方支持文章](https://support.kaspersky.com/15163#block2)来了解如何把程序加入信任名单。

#### 允许程序通过迈克菲 (McAfee) {id=firewall-whitelist-mcafee}

请参阅迈克菲的[官方知识库文章](https://service.mcafee.com/?articleId=TS100813&page=shell&shell=article-view)。

### 我已经允许 osu! 通过防火墙了，但是我仍然没办法连接游戏！ {id=sign-in-unknown-cause}

**我们需要更多信息才能帮助你解决问题。**

如果你仍然不能连接到游戏服务器，请发邮件给 [accounts@ppy.sh](mailto:accounts@ppy.sh) 来申请帮助。请提供以下信息来帮助我们快速解决你的问题：

- 使用 `tracert` 命令，获取对 `cho.ppy.sh`，`osu.ppy.sh` 和 `m1.ppy.sh` 的路由追踪日志（可参照下方的指示）。
- 游戏安装目录，`Logs` 文件夹内的 `network.log`。

#### 进行路由追踪来排除网络问题 {id=traceroute}

**路由追踪可以让我们检测你与游戏服务器中间到底出了什么问题。**

你可以在命令提示符中使用 `tracert` 命令来进行路由追踪。

如果要打开命令提示符，请按住键盘上的 `Windows` 键，然后点击 `R`。然后在弹出的窗口中输入 `cmd`，然后按下 `确定`。详细信息请参阅[这篇 Windows 知识库文章](https://support.microsoft.com/zh-cn/topic/%E5%A6%82%E4%BD%95%E4%BD%BF%E7%94%A8-tracert-%E8%A7%A3%E5%86%B3-windows-%E4%B8%AD%E7%9A%84-tcp-ip-%E9%97%AE%E9%A2%98-e643d72b-2f4f-cdd6-09a0-fd2989c7ca8e)。

随后，你就可以把指令输入进去了。举个例子，`tracert cho.ppy.sh` 会对你电脑与游戏服务器的通信进行追踪。这可能需要一分钟或者几分钟，具体依据你的网络情况而定。

我们需要对 `cho.ppy.sh`、`osu.ppy.sh` 和 `m1.ppy.sh` 的路由追踪日志。

如果要复制追踪结果，点击运行结果的底部，向上拖动，结果都变白之后即可右键复制。

### 在线游玩时，我收到了服务器端的错误！ {id=server-side-error}

**游戏服务器可能正面临问题。**

请看看 [osu! 服务器状态页面](https://status.ppy.sh/ "osu! 服务器状态")来确定是不是服务器出现了故障。

**这也可能因为你使用了共享网络连接，代理服务器或者 VPN。**

尝试换一个代理服务器或 VPN 服务提供商，如果不奏效，请发邮件给 [accounts@ppy.sh](mailto:accounts@ppy.sh) 来请求帮助。

如果你没有用过那些东西，那么请直接请求帮助，你的连接可能被我们误封禁了。

Tor 网络或通过 Tor 的连接将无法连接大部分游戏服务器。我们对于 Tor 的使用持中立意见，但是我们这么做只是为了阻止坏人使用 Tor 干坏事。

### 我成为了 osu! 支持者，但是游戏里我的用户名不是黄色的！ {id=white-username}

**在游戏客户端当中，你自己的用户名永远是白色的。其他玩家可以看到你闪闪发光的黄色名字！**

这可以帮助你区分你自己和其他人的消息，目前无法禁用这个功能。

## 性能 {id=performance}

### 我的电脑运行 osu! 会过热，怎么让它冷下来？ {id=overheating}

**最简单的方式就是改变你的帧数限制设置。**

若要检查你目前的帧数限制，请参阅如下步骤：

1. 打开 osu!。
2. 在主页点击 `Options`，或者按下 `Ctrl` + `O`。
3. 直接输入 `frame` 可以看到你目前的帧数限制设置。

如果你的设置是`无限制`，把设置更改为其他三个可能会有明显的效果。

`垂直同步`会让游戏运行在你的显示器帧率下，而 `Power Saving` 会让游戏运行在显示器的两倍帧率下，`Optimal` 会让游戏运行在显示器的八倍帧率下，最高 960Hz。

如果你的电脑会过热，我们建议选择`垂直同步`。

你还可以了解一些控制处理器和显卡风扇转速的程序。

### 我的电脑会在玩 osu! 的时候自动重启！ {id=reboot-overheating}

**多数时候，这是因为你的电脑一个或多个部件过热了，大概率是你的显卡或者处理器。**

如果要搞清楚原因，我们建议你使用检测显卡和处理器温度的程序，并且在游戏运行的时候实时关注温度值。[Speccy](https://www.ccleaner.com/speccy) 是一个很好的选择。

如果你注意到 GPU 或 CPU 温度高于 90°C 或 194°F，你的电脑就已经很热了，并极为接近多数机器自动重启的上限。

### 我的电脑没有过热，但是游戏仍然会导致我的电脑重启！ {id=reboot-faulty-device}

**过热后的第二大原因就是声卡问题，第三大原因是你的外设出现问题。**

旧一些的声卡驱动或软件可能会导致 osu! 出现问题，但是对大多数人来讲这不属于大问题。尝试更新你的声卡或主板驱动。

你还可以试一下拔除连接到你电脑的一些 USB 外设，尤其是那些处理声音信号的设备。耳机、USB 声卡或它们连接到的接口一旦坏掉，可能会导致问题。

### 试过以上方法后还是重启！ {id=reboot-nothing-helps}

如果你已经试过所有方式，但问题仍然出现，请联系 [accounts@ppy.sh](mailto:accounts@ppy.sh) 来请求帮助。请附上有用的信息，比如你的电脑配置、型号（如果适用）和你认为可能有助于解决问题的其他信息。

### 玩 osu!mania 的时候，我不能一次性按下 n 个按钮！ {id=keyboard-ghosting}

**这是因为大部分键盘都存在的*按键冲突*问题。**

按键冲突这个问题有些复杂，本质是你的键盘制作工艺上的缺陷，导致它不能发送一次性按下多个键的信号。

大部分的新键盘可以一次性按下 **6** 个按键，不管按下的是什么键。

你可以用 [drakeirving 的键盘冲突小工具](https://drakeirving.github.io/MultiKeyDisplay)来检查自己的键盘一次性可以正常按下多少个键。

虽然 osu!mania 的谱面上架准则 (RC) 已经规定谱面不能出现同时按下 6 个键以上的配置，但如果你的键盘比较老旧，按键冲突的问题可能还是会影响到你。

治标治本的解决方案只有购买一个能同时处理更多按键的新键盘。如果那样的话，你可以关注*全键无冲*的键盘。这种键盘基本不会有按键冲突的问题，但是一般比较昂贵。

### 玩 osu! 的时候，我的光标疯跑！ {id=crazy-cursor}

**最常见的原因是你游戏内的鼠标灵敏度设置的太高了！**

光标乱跳、奇怪的移动尤为典型，尤其是在你使用数位板时。

大部分情况下，把游戏内的灵敏度更改为 1.0x（默认设置）会解决这个问题，但是可能会导致你的鼠标移动比较缓慢。我们建议你打开游戏中的 `原始输入 (Raw Input)` 来让 osu! 直接通过设备驱动来调整灵敏度。

如果你使用鼠标，你还可以尝试更改你的鼠标 *DPI* 设置。更高的 DPI 设置能让游戏收到更准确的定位，同时可能会平滑掉迟缓的感觉。

我们还建议你检查自己的外设是否足够干净。如果有毛发或灰尘挡住鼠标的光学传感器，可能会导致光标移动很怪。

#### 启用原始输入 (Raw Input) {id=raw-input}

**原始输入 (Raw Input) 会让游戏使用更加干净直接的方式获取来自外设的信息，这通常会提高设备输出的精度。**

若要打开原始输入 (Raw Input)，请参阅以下步骤：

1. 打开 osu!。
2. 点击主页的 `Options`，或者按下 `Ctrl` + `O`。
3. 直接输入 `raw`，即可跳转到对应的设置区域。
4. 点击 `原始输入 (Raw Input)` 来将它打开。

### 我使用数位板，我感觉我的光标移动太慢了！ {id=slow-cursor}

**你可以设置更小的数位板映射区域来解决问题。**

在大多数 Wacom 数位板上，你可以在 `Wacom 数位板属性`的`映射`标签页中完成操作。

![Wacom 数位板属性屏幕显示映射选项](img/tablet-area.png "Wacom 设备更改数位板映射区域的界面")

你需要咨询你的设备生产商来了解如何修改你的设备的映射区域。

所有 osu! 品牌的数位板均支持原始输入 (Raw Input)，可以直接使用游戏中的鼠标灵敏度来轻松调整光标灵敏度。

### 玩游戏的时候，我的电脑出现卡顿！ {id=lag-spikes}

**游戏出现卡顿有多种原因，可能是因为软件或过期的驱动，也可能只是显示设置调整过高。**

#### 玩 osu! 的时候，你的后台有很多程序吗? {id=lag-too-many-programs}

**部分程序可能会与 osu! 争抢系统资源，造成游戏体验不佳。**

尝试关闭一部分或者全部玩游戏时不需要的程序。

#### 你的电脑安全吗？ {id=lag-infected-pc}

**病毒、后门程序、恶意程序或其他你可能不想要的程序可能会导致严重的性能问题，哪怕是很高级的电脑。**

保证电脑安全可能会改善问题。尝试在电脑上运行病毒查杀来解决这些问题。

#### 你的电脑用多久了？ {id=lag-old-pc}

**非常久的电脑可能使用更旧的 Intel 集显，导致无法运行游戏，就算设置已经全部改成最低。**

解决问题的唯一方法恐怕只有升级电脑了。

#### 你的驱动是最新版本吗？ {id=lag-old-driver}

**过期的驱动可能导致严重的性能问题。如果可以，请一直保持驱动为最新状态。**

显卡驱动对 osu! 尤其重要，声卡驱动同样重要。

#### 你使用一个 USB 集线器连接了很多 USB 设备吗？ {id=lag-faulty-usb}

**只使用一个插孔并用集线器连接太多设备可能会导致输入延迟过高，这通常很难以发现，但是会对游戏造成较大影响。**

一个简单的解决办法是：拔掉集线器，如果问题解决，你就知道问题所在啦！

#### 你正在使用窗口化或全屏无边框模式运行 osu! 吗？ {id=lag-windowed}

**窗口化和全屏无边框均会带来*可观*的输入延迟。若要保证游戏体验，请使用全屏模式。**

#### 你的游戏只是卡顿一小会吗？ {id=lag-background-process}

**有时候，电脑上的后台程序可能会一次性处理过多数据，这可能会让机器暂时力不从心。**

在玩游戏之前关闭不必要的后台程序通常可以解决问题。P2P 和种子下载程序通常影响最大，尤其是当它与 osu! 安装在同一块硬盘上的时候。

#### 游戏最近更新了吗？ {id=lag-after-update}

**有些时候，更新客户端后可能会导致客户端莫名卡顿。**

如果你有很多谱面，在怪罪更新之前，请等待 10-15 分钟。osu! 可能正在后台重新计算谱面难度。

如果等待后无法解决问题，重启可能会有帮助。

**如果游玩时你仍然会遇到卡顿停顿问题，请参阅 [osu! 性能问题检查](/wiki/Performance_troubleshooting)来获知更多问题和可能的解决方案。**
